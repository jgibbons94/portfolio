Novice to Ninja Chapters 5-7

  • Objects are structures of data and functions
  • Objects are always mutable, even if they are declared with const
  • Objects can be declared literally or with a constructor
  • this refers to the object the current function is attached to.
  • Namespaces are objects.
  • Namespaces can be functional. Look at the Math namespace for an example.
  • JSON is a serialization format.
  • Date is used to get timestamp information
  • Date was invented before getter functions could be attached to member variables
  • RegExp is used for matching.
  • RegExp objects can be declared like in ed.
  • There was nothing in Chapter 5 I did not already know.
  • DOM means Document Object Model
  • DOM is useful for manipulating html and xml data.
  • DOM represents everything as a tree.
  • DOM is accessed through the window.document object.
  • window is the root "this" in a browser, so DOM can be accessed through the document object
  • document provides really long names for accessing everything. It's annoying
  • I just use document.querySelector and document.querySelectorAll to access what I need.
  • DOM can manipulate CSS-relevant properties.
  • There is nothing in chapter 6 I did not already know.
  • User inputs are handled with event listeners.
  • Ways to set up event listeners include specifying them in the html, setting them up in objects, and using the "addEventListener" function of all nodes.
  • Why does a recent javascript book use alert in its examples? Nobody who uses a web browser likes alert. Nobody. Likes. alert.
  • There is nothing in chapter 7 I do not already know. I don't blame the course, I blame the university for not providing a way for me to "test out" of something I did as a student from 2011-2013 and professionally from 2015-2018.
