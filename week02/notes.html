Novice to Ninja Chapters 2-4

  • alert() and prompt() are two very annoying functions, and I'm not sure why we continue to teach them to novice JavaScript programmers. They cause popdown messages that block the user's window and waste the user's time. Use well-placed divs instead.
  • JavaScript loops are very similar to all other loops in every procedural programming language ever. I prefer the functional map, foreach, and reduce, because that allows me to avoid array index out of bounds errors.
  • Structures included within javascript include arrays, sets, and maps.
    • arrays are lists declared between [ and ]. They behave like pythonic lists, except there are no list comprehensions in javascript yet.
    • Sets are lists of unique elements. They can be created by making a new Set from an array of items. They are just like pythonic lists. There is no shorthand way to declare them in json.
    • Maps are like pythonic dicts, except there are no list comprehensions in javascript yet. They map keys to values.
  • It is possible to match elements in an array Haskell-style. Neat!
