Special Function Types in JavaScript

  • Immediately Invoked Function Expressions
    • Anonymous functions.
    • Wrap everything in a limited scope.
    • Why not just wrap code in braces?
  • Async functions
    • asynchronous
    • declared with async keyword
    • called from normal functions with await keyword
    • can call other asynchronous functions directly with async keyword
    • they can be defined anonymously as lambdas preceded by the async keyword.
  • Generators
    • declared with function*
    • yeild instead of return.
    • continue where they left off on each call.
    • impure functions
    • I don't know if there's a way to define them anonymously.
